# Full Stack Saas Spa - Project Structure

This project has been converted from a static React site to a full-stack MERN application.

## Structure

*   **/frontend**: The React client application.
*   **/backend**: The Express/Node.js REST API.

## setup

### Backend

1.  Navigate to `backend` folder.
2.  Run `npm install`.
3.  Create a `.env` file with `MONGO_URI` and `JWT_SECRET`.
4.  Run `npm start` (or `node server.js`).

### Frontend

1.  Navigate to `frontend` folder.
2.  Run `npm install`.
3.  Run `npm run dev`.
